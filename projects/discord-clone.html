Discord UI Clone

UI web ini dirancang untuk meniru tampilan antarmuka Discord.gg dengan nuansa visual yang modern dan interaktif. Proyek ini dibuat menggunakan HTML sebagai struktur dasar, dipadukan dengan Tailwind CSS untuk memberikan styling yang fleksibel serta konsisten. JavaScript turut digunakan untuk menghadirkan elemen dinamis dan responsif sehingga pengalaman pengguna terasa lebih hidup. Melalui kombinasi teknologi tersebut, UI ini berhasil menampilkan simulasi desain Discord yang menarik sekaligus menjadi sarana pembelajaran dalam memahami sistem antarmuka web profesional.
